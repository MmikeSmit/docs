介绍用户如何开始使用您的产品。他们需要做的第一件事是什么？是否有任何先决条件？

快速入门应能让用户从零开始使用您的产品。他们将获得一个小成果，并了解可以用产品做些什么。

## 先决条件

在开始之前，您必须具备：

- 要求一（例如，Node.js 18+、一个免费账户、一个 API 密钥）
- 要求二（例如，兼容的设备、兼容的浏览器、兼容的操作系统）

## 开始使用

<Steps>
  <Step title="安装">
    介绍如何安装您的产品或注册账户。

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="配置">
    介绍首次使用前所需的任何设置或配置。

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="运行">
    展示用户首次操作以查看其运行效果。

    ```bash
    your-cli start
    ```
  </Step>
</Steps>
