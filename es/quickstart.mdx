Describe cómo alguien empieza a usar tu producto. ¿Qué es lo primero que debe hacer? ¿Hay algún requisito previo?

Un inicio rápido debe llevar a alguien desde cero hasta usar tu producto. Conseguirán un logro rápido y una idea de lo que pueden lograr.

## Requisitos previos

Antes de empezar, debes tener:

- Requisito uno (por ejemplo, Node.js 18+, una cuenta gratuita, una clave API)
- Requisito dos (por ejemplo, un dispositivo compatible, un navegador compatible, un sistema operativo compatible)

## Empezar

<Steps>
  <Step title="Instalar">
    Describe cómo instalar tu producto o registrarse.

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="Configurar">
    Describe cualquier configuración necesaria antes del primer uso.

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="Ejecutar">
    Muestra lo primero que hace el usuario para verlo funcionar.

    ```bash
    your-cli start
    ```
  </Step>
</Steps>
