La inversión total es el margen inicial destinado a este copy trading, expresado en USDC.

Al introducirlo, ten en cuenta:

- El importe no puede ser inferior al mínimo del sistema.
- El importe no puede superar tu saldo disponible en USDC.
- La página muestra tu saldo actual.
- Puedes usar el control deslizante para seleccionar rápidamente 25%, 50%, 75% o 100%.

<Info>
El mínimo a invertir lo define la configuración del sistema. La página avisa cuando el importe es demasiado bajo y muestra el mínimo requerido.
</Info>

## Saldo y comisión de activación

Si al crear el copy trading hay que abrir una nueva subcuenta de copy trading, el sistema cobra una comisión de activación adicional de 1 USDC desde la cartera principal.

La página muestra un aviso cuando se requiere esta comisión. Afecta al importe máximo invertible, así que aunque tengas suficiente USDC en la cartera, debes reservar la comisión de activación.

Si ya tienes una cuenta de copy trading reutilizable, normalmente no tendrás que volver a pagar esta comisión.
