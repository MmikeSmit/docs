---
title: "Razones comunes por las que no se puede crear"
description: "Solucionar problemas al crear copy trading o cuando el botón no está disponible."
mode: "wide"
---

Si el botón de **Copy trading** no está disponible, suele deberse a una de estas razones:

| Razón | Descripción |
| --- | --- |
| Billetera no conectada | Debes conectar tu billetera antes de crear un copy trading. |
| Dirección de billetera inválida | La dirección del trader debe ser una dirección EVM válida. |
| Monto de inversión demasiado bajo | La inversión total está por debajo del mínimo del sistema. |
| Saldo insuficiente | El USDC disponible no alcanza para cubrir el monto invertido y la posible tarifa de activación. |
| La billetera objetivo no admite copy trading | Algunas billeteras pueden tener deshabilitado el copy trading. |
| Límite de subcuentas alcanzado | Llegaste al límite de subcuentas de copy trading; necesitas terminar un copy trading existente antes de crear uno nuevo. |
