Tras crear el copy trading, entra a la página de copy trading en tu perfil para ver:

- Patrimonio del copy trading
- PNL de 7 días
- Margen disponible
- Ganancias y pérdidas
- Ganancias y pérdidas no realizadas
- Reglas de copy trading activas
- Historial de copy trading

## Gestionar el copy trading

En la página de copy trading de tu perfil puedes gestionar las reglas de copy trading que ya creaste.

Operaciones comunes:

| Acción | Descripción |
| --- | --- |
| Detalles | Ver los detalles de la regla de copy trading. |
| Cerrar | Detener el copy trading actual. |
| Aumentar margen | Añadir margen a la cuenta de copy trading. |
| Compartir datos | Compartir el desempeño del copy trading. |
| Copiar de nuevo | Crear un nuevo copy trading sobre la misma billetera objetivo. |
| Eliminar | Eliminar reglas cerradas o inactivas. |

<Warning>
Cerrar el copy trading y eliminar la regla son acciones distintas. Cerrar sirve para dejar de seguir; eliminar suele usarse para limpiar registros inactivos.
</Warning>
