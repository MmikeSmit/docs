Una vez creado correctamente, ve a la página de copy trading del perfil personal para consultar:

- Capital del copy trading
- PnL de 7 días
- Margen disponible
- Ganancias y pérdidas
- PnL no realizado
- Reglas de copy trading activas
- Historial de copy trading

## Gestionar copy tradings

En la página de copy trading del perfil personal puedes gestionar las reglas creadas.

Operaciones habituales:

| Operación | Descripción |
| --- | --- |
| Detalles | Ver los detalles de la regla de copy trading. |
| Cerrar | Detener el copy trading actual. |
| Añadir margen | Aportar margen a la cuenta de copy trading. |
| Compartir datos | Compartir el rendimiento del copy trading. |
| Volver a copiar | Crear un nuevo copy trading sobre la misma cartera objetivo. |
| Eliminar | Eliminar reglas cerradas o inactivas. |

<Warning>
Cerrar un copy trading y eliminar una regla son operaciones distintas. Cerrar sirve para detener el seguimiento; eliminar suele usarse para limpiar registros inactivos.
</Warning>
