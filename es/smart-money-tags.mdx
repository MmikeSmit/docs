## Etiquetas de perfil de billetera

Junto al avatar de la tarjeta de la billetera se muestra un grupo de etiquetas del sistema que resumen rápidamente las características de operación de esa billetera. No son las etiquetas de favoritos que agregas manualmente, sino un perfil generado a partir de los datos de la billetera.

Las etiquetas comunes se pueden entender en cuatro categorías:

### Tamaño de cuenta

Se usa para clasificar el volumen de la billetera según el tamaño de la posición actual.

| Etiqueta | Significado |
| --- | --- |
| Ballena | Billetera con una posición actual muy grande. |
| Tiburón | Billetera con una posición actual grande. |
| Delfín | Billetera con una posición actual mediana. |
| Pez pequeño | Billetera con una posición actual pequeña. |

### Preferencia de activos

Se usa para identificar qué tipo de activos opera principalmente la billetera.

| Etiqueta | Significado |
| --- | --- |
| Majors | Opera con más frecuencia activos principales como BTC y ETH. |
| Altcoins | Opera con más frecuencia activos no principales y de alta volatilidad. |
| TradFi | Opera con más frecuencia activos relacionados con finanzas tradicionales. |
| Mixto | Opera varios tipos sin una preferencia única clara. |

### Preferencia de dirección

Se usa para determinar si la billetera tiende más a posiciones largas, cortas o si está relativamente equilibrada.

| Etiqueta | Significado |
| --- | --- |
| Sesgo largo | Mantiene con más frecuencia posiciones largas. |
| Sesgo corto | Mantiene con más frecuencia posiciones cortas. |
| Equilibrada | Distribución entre largos y cortos relativamente equilibrada. |

### Horizonte de trading y marcas de desempeño

Se usa para identificar el ritmo de trading y si hay características destacadas en el desempeño reciente.

| Etiqueta | Significado |
| --- | --- |
| Ultra corto plazo | Tiende a operar con alta frecuencia. |
| Corto plazo | Tiende a mantener posiciones por períodos cortos. |
| Mediano y largo plazo | Tiende a mantener posiciones por períodos más largos. |
| Rey del win rate | Desempeño destacado en porcentaje de aciertos. |
| Crecimiento estable | Desempeño más estable en plazos largos. |
| Máquina de ganancias | Actividad de trading y rentabilidad destacadas. |

Por ejemplo, el **Equilibrada / Sesgo largo / Corto plazo** que aparece en la captura indica que la distribución de la billetera entre largos y cortos está relativamente equilibrada, recientemente se inclina más hacia largos y su ritmo de trading es de corto plazo. Puedes usar primero estas etiquetas para juzgar si encaja con tus preferencias de copy trading y luego entrar al detalle para ver posiciones e historial.

### Cómo se determina el tamaño de cuenta

El tamaño de cuenta se clasifica según el monto de la posición actual:

| Etiqueta | Criterio |
| --- | --- |
| Ballena | Posición mayor a 1.000.000 USDC |
| Tiburón | Posición entre 100.000 y 1.000.000 USDC |
| Delfín | Posición entre 50.000 y 100.000 USDC |
| Pez pequeño | Posición entre 10.000 y 50.000 USDC |

Esta etiqueta sirve principalmente para juzgar el volumen de la billetera. A mayor volumen, mejor suele representar un comportamiento real de capital; en billeteras pequeñas, el porcentaje de rentabilidad puede amplificarse fácilmente con unas pocas operaciones.

### Cómo se determinan las marcas de desempeño

Las marcas de desempeño señalan cuentas con desempeño reciente destacado:

| Etiqueta | Criterio |
| --- | --- |
| Rey del win rate | Win rate mayor al 80% y más de 60 operaciones abiertas en 30 días |
| Crecimiento estable | Ganancias estables durante más de medio año y más de 30 operaciones abiertas en 30 días |
| Máquina de ganancias | Win rate mayor al 60% y más de 300 operaciones abiertas en 30 días |

Estas marcas solo indican que la billetera cumple las condiciones de datos correspondientes; no garantizan que siga siendo rentable. Para decidir si copiarla, también revisa las posiciones actuales, la caída máxima, el apalancamiento y los activos operados.
