---
title: "無法建立的常見原因"
description: "排查跟單建立失敗或按鈕不可用。"
mode: "wide"
---

如果 **跟單** 按鈕不可用，通常是以下原因之一：

| 原因 | 說明 |
| --- | --- |
| 未連接錢包 | 需要先連接錢包才能建立跟單。 |
| 錢包地址無效 | 帶單錢包地址必須是有效的 EVM 地址。 |
| 投入金額過低 | 總投入低於系統最低要求。 |
| 餘額不足 | 可用 USDC 不足以涵蓋投入金額與可能的啟用費。 |
| 目標錢包不可跟單 | 部分錢包可能已關閉跟單。 |
| 子帳戶數量達到上限 | 已達到跟單子帳戶上限，需要先結束一筆現有跟單再建立新的跟單。 |
