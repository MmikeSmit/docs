建立成功後，進入個人中心的跟單頁面，可以查看：

- 跟單權益
- 7 日盈虧
- 可用保證金
- 盈虧
- 未實現盈虧
- 目前活躍的跟單規則
- 跟單歷史

## 管理跟單

在個人中心的跟單頁面，可以管理已經建立的跟單規則。

常見操作包括：

| 操作 | 說明 |
| --- | --- |
| 詳情 | 查看跟單規則詳情。 |
| 關閉 | 停止目前跟單。 |
| 增加保證金 | 為跟單帳戶增加保證金。 |
| 分享資料 | 分享跟單表現。 |
| 再次跟單 | 基於同一目標錢包再次建立跟單。 |
| 刪除 | 刪除已關閉或非活躍規則。 |

<Warning>
關閉跟單與刪除規則是不同操作。關閉用於停止跟隨；刪除通常用於清理非活躍紀錄。
</Warning>
