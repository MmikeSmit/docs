## Wallet profile tags

Wallet cards show system-generated profile tags beside the wallet avatar. These tags summarize the wallet's trading characteristics. They are not manual favorite labels; they are generated from wallet data.

Common tags can be read in four groups:

### Account size

Used to classify wallet scale by current position exposure.

| Tag | Meaning |
| --- | --- |
| Whale | Very large current position exposure. |
| Shark | Large current position exposure. |
| Dolphin | Medium current position exposure. |
| Fish | Smaller current position exposure. |

### Market preference

Used to identify what types of markets the wallet mainly trades.

| Tag | Meaning |
| --- | --- |
| Majors | More often trades BTC, ETH, and other major markets. |
| Alts | More often trades higher-volatility alternative markets. |
| TradFi | More often trades traditional-finance-related markets. |
| Mixed | Trades across multiple market types without one dominant preference. |

### Directional bias

Used to show whether the wallet more often takes long, short, or balanced exposure.

| Tag | Meaning |
| --- | --- |
| Long-biased | More often holds long exposure. |
| Short-biased | More often holds short exposure. |
| Balanced | Long and short exposure are relatively balanced. |

### Holding period and performance markers

Used to identify trading tempo and standout recent behavior.

| Tag | Meaning |
| --- | --- |
| Scalping | More short-frequency trading. |
| Short-term | More short holding periods. |
| Medium-term | More medium holding periods. |
| Win-rate Hunter | Standout win-rate behavior. |
| Steady Compounder | More stable long-period performance. |
| Profit Machine | Standout activity and profitability. |

For example, **Balanced / Long-biased / Short-term** means the wallet is relatively balanced across long and short exposure overall, has recently leaned more long, and tends to trade on a shorter time frame. Use these tags to decide whether the wallet matches your copy trading preference before opening the details page.

### Account size criteria

Account size is based on current position exposure:

| Tag | Criteria |
| --- | --- |
| Whale | Position exposure above 1,000,000 USDC |
| Shark | Position exposure from 100,000 to 1,000,000 USDC |
| Dolphin | Position exposure from 50,000 to 100,000 USDC |
| Fish | Position exposure from 10,000 to 50,000 USDC |

This tag helps estimate wallet scale. Larger wallets usually reflect more meaningful capital behavior. Smaller wallets can show inflated percentage returns from only a few trades.

### Performance marker criteria

Performance markers highlight accounts with notable recent trading behavior:

| Tag | Criteria |
| --- | --- |
| Win-rate Hunter | Win rate above 80% and more than 60 opened trades in the last 30 days |
| Steady Compounder | Profitable for more than six months and more than 30 opened trades in the last 30 days |
| Profit Machine | Win rate above 60% and more than 300 opened trades in the last 30 days |

These markers only mean the wallet meets the data condition. They do not guarantee future profitability. Before following, also review current positions, drawdown, leverage, and traded markets.
