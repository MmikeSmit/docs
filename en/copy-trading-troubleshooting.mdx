---
title: "Common Creation Issues"
description: "Troubleshoot failed copy trading creation or disabled buttons."
mode: "wide"
---

If the **Copy Trade** button is disabled, it is usually caused by one of the following reasons:

| Reason | Explanation |
| --- | --- |
| Wallet not connected | You need to connect a wallet before creating a copy trading rule. |
| Invalid wallet address | The lead wallet address must be a valid EVM address. |
| Allocation too low | Total allocation is below the system minimum. |
| Insufficient balance | Available USDC is not enough to cover the allocation and possible activation fee. |
| Target wallet cannot be copied | Some wallets may have copy trading disabled. |
| Sub-account limit reached | You have reached the copy trading sub-account limit. Close an existing rule before creating a new one. |
