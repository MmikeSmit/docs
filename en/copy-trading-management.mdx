After a rule is created, open the Copy Trading page in Profile to view:

- copy trading equity
- 7-day PnL
- available margin
- PnL
- unrealized PnL
- current active copy trading rules
- copy trading history

## Manage copy trading

On the Copy Trading page in Profile, you can manage existing rules.

Common actions include:

| Action | Description |
| --- | --- |
| Details | View rule details. |
| Close | Stop the current copy trading rule. |
| Add margin | Add margin to the copy trading account. |
| Share data | Share copy trading performance. |
| Copy again | Create another rule based on the same lead wallet. |
| Delete | Delete closed or inactive rules. |

<Warning>
Closing a rule and deleting a rule are different actions. Close stops the rule from following; delete is usually used to clean up inactive records.
</Warning>
