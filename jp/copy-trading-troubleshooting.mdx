---
title: "作成できない一般的な原因"
description: "コピートレード作成失敗やボタン利用不可のトラブルシューティング。"
mode: "wide"
---

**コピートレード** ボタンが利用できない場合は、通常次のいずれかが原因です。

| 原因 | 説明 |
| --- | --- |
| ウォレット未接続 | コピートレードを作成するには、まずウォレットを接続する必要があります。 |
| 無効なウォレットアドレス | トレーダーのウォレットアドレスは有効な EVM アドレスである必要があります。 |
| 投入金額が低すぎる | 総投入額がシステムの最低要件を下回っています。 |
| 残高不足 | 利用可能な USDC が投入金額および発生する可能性のあるアクティベーション料をカバーするのに足りません。 |
| 対象ウォレットがコピートレード不可 | 一部のウォレットはコピートレードを停止している場合があります。 |
| サブアカウント数が上限に到達 | コピートレード用サブアカウントが上限に達しているため、既存のコピートレードを 1 件終了させてから新規作成する必要があります。 |
