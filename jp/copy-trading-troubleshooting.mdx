---
title: "作成できない場合のよくある原因"
description: "コピートレード作成失敗やボタンが使用不可となる原因の切り分け。"
mode: "wide"
---

**コピートレード** ボタンが使用不可となる場合、通常は次のいずれかが原因です。

| 原因 | 説明 |
| --- | --- |
| ウォレット未接続 | コピートレードを作成するには、まずウォレットを接続する必要があります。 |
| ウォレットアドレスが無効 | フォロー対象ウォレットのアドレスは有効な EVM アドレスである必要があります。 |
| 投入金額が低すぎる | 総投入額がシステムの最低要件を下回っています。 |
| 残高不足 | 利用可能な USDC が投入金額と発生し得るアクティベーション費を満たしません。 |
| 対象ウォレットがコピートレード不可 | 一部のウォレットはコピートレードを停止している場合があります。 |
| サブアカウント数が上限に到達 | コピートレードサブアカウントが上限に達しているため、既存のコピートレードを 1 つ終了してから新たに作成する必要があります。 |
