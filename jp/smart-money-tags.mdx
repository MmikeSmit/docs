## ウォレットプロファイルタグ

ウォレットカードのアバターの横には、そのウォレットの取引特性をすばやく把握するためのシステムタグが表示されます。これらはユーザーが手動で追加するお気に入りラベルではなく、ウォレットデータをもとに生成されるプロファイルです。

よく使われるタグは次の4つのカテゴリーに分けて理解できます。

### アカウント規模

現在のポジション規模に応じてウォレットの大きさを区別するために使用します。

| タグ | 意味 |
| --- | --- |
| クジラ | 現在のポジション規模が非常に大きいウォレット。 |
| サメ | 現在のポジション規模が大きいウォレット。 |
| イルカ | 現在のポジション規模が中程度のウォレット。 |
| 小魚 | 現在のポジション規模が小さいウォレット。 |

### 銘柄の好み

ウォレットが主にどのような種類の銘柄を取引しているかを判断するために使用します。

| タグ | 意味 |
| --- | --- |
| メジャー | BTC、ETH などのメジャー銘柄をより多く取引する。 |
| アルトコイン | ボラティリティの高い非メジャー銘柄をより多く取引する。 |
| TradFi | 伝統的な金融関連の銘柄をより多く取引する。 |
| ミックス | 複数の種類を取引しており、明確な単一の好みがない。 |

### 方向性の好み

ウォレットがロング寄りか、ショート寄りか、それともロング・ショートが比較的均衡しているかを判断するために使用します。

| タグ | 意味 |
| --- | --- |
| ロング寄り | ロングポジションを保有することが多い。 |
| ショート寄り | ショートポジションを保有することが多い。 |
| バランス型 | ロングとショートの分布が比較的均衡している。 |

### 取引サイクルとパフォーマンスマーカー

取引のテンポと、最近のパフォーマンスに目立つ特徴があるかどうかを判断するために使用します。

| タグ | 意味 |
| --- | --- |
| 超短期 | 短い頻度での取引が多い。 |
| 短期 | 短い保有期間の傾向。 |
| 中長期 | 比較的長い保有期間の傾向。 |
| 勝率ハンター | 勝率のパフォーマンスが際立っている。 |
| 安定成長 | 長期的なパフォーマンスがより安定している。 |
| 利益マシン | 取引の活発さと収益パフォーマンスが際立っている。 |

たとえばスクリーンショットの **バランス型 / ロング寄り / 短期** は、このウォレットがプロファイル上はロングとショートの分布が比較的均衡しており、最近の方向性はややロング寄りで、取引のテンポは短期寄りであることを示しています。まずこれらのタグで自分のコピートレード傾向に合うかどうかを判断し、それから詳細ページに進んでポジションや取引履歴を確認できます。

### アカウント規模の判定方法

アカウント規模は、現在のポジション金額に応じて区分されます。

| タグ | 判定基準 |
| --- | --- |
| クジラ | ポジションが 100 万 USDC 超 |
| サメ | ポジションが 10 万〜100 万 USDC |
| イルカ | ポジションが 5 万〜10 万 USDC |
| 小魚 | ポジションが 1 万〜5 万 USDC |

このタグは主にウォレットの規模を判断するためのものです。規模が大きいほど、通常は実際の資金の動きをより正確に表しますが、規模が小さいウォレットでは、わずかな取引で収益率が大きく見えやすくなります。

### パフォーマンスマーカーの判定方法

パフォーマンスマーカーは、最近の取引パフォーマンスが際立っているアカウントを示すために使用されます。

| タグ | 判定基準 |
| --- | --- |
| 勝率ハンター | 勝率 80% 超、かつ 30 日間のエントリー数が 60 を超える |
| 安定成長 | 半年以上にわたり安定して利益を出し、かつ 30 日間のエントリー数が 30 を超える |
| 利益マシン | 勝率 60% 超、かつ 30 日間のエントリー数が 300 を超える |

これらのマーカーは、ウォレットが該当するデータ条件を満たしていることを示すだけで、今後も利益を出し続けることを保証するものではありません。フォローするかどうかを判断する際は、現在のポジション、ドローダウン、レバレッジ、取引銘柄も併せて確認してください。
