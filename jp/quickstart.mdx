ユーザーが製品を使い始める方法を説明します。最初に何をする必要がありますか？前提条件はありますか？

クイックスタートは、ユーザーをゼロから製品を使える状態へ導くものです。ちょっとした成功体験と、製品でできることのイメージを得られるようにします。

## 前提条件

開始する前に、以下が必要です：

- 要件 1（例：Node.js 18+、無料アカウント、API キー）
- 要件 2（例：対応デバイス、対応ブラウザ、対応 OS）

## はじめる

<Steps>
  <Step title="インストール">
    製品のインストール方法またはサインアップ方法を説明します。

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="設定">
    初回利用前に必要なセットアップや設定を説明します。

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="実行">
    動作確認のためにユーザーが最初に行う操作を示します。

    ```bash
    your-cli start
    ```
  </Step>
</Steps>

<Tip>
  ヘルプを得られる手段を用意しましょう。サポートページへのリンク、カスタマーサポートとのチャット、または製品フォーラムなどです。

  例：お困りですか？ [support@yourcompany.com](mailto:support@yourcompany.com) までお問い合わせください。
</Tip>
