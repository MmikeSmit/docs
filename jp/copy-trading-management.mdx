作成に成功したら、マイページのコピートレード画面で次の情報を確認できます。

- コピートレード権益
- 7 日損益
- 利用可能証拠金
- 損益
- 未実現損益
- 現在アクティブなコピートレードルール
- コピートレード履歴

## コピートレードを管理する

マイページのコピートレード画面では、すでに作成したコピートレードルールを管理できます。

主な操作は次のとおりです。

| 操作 | 説明 |
| --- | --- |
| 詳細 | コピートレードルールの詳細を表示します。 |
| 停止 | 現在のコピートレードを停止します。 |
| 証拠金追加 | コピートレードアカウントに証拠金を追加します。 |
| データ共有 | コピートレードのパフォーマンスを共有します。 |
| 再度コピートレード | 同じ対象ウォレットを基にコピートレードを再作成します。 |
| 削除 | 停止済みまたは非アクティブなルールを削除します。 |

<Warning>
コピートレードの停止とルールの削除は異なる操作です。停止はフォローを止めるため、削除は通常非アクティブな記録を整理するために使用します。
</Warning>
