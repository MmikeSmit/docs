作成成功後、マイページのコピートレードページに進むと、次の情報を確認できます。

- コピートレードエクイティ
- 7 日損益
- 利用可能証拠金
- 損益
- 未実現損益
- 現在アクティブなコピートレードルール
- コピートレード履歴

## コピートレードを管理する

マイページのコピートレードページでは、作成済みのコピートレードルールを管理できます。

よく使う操作には次のものがあります。

| 操作 | 説明 |
| --- | --- |
| 詳細 | コピートレードルールの詳細を確認します。 |
| 停止 | 現在のコピートレードを停止します。 |
| 証拠金追加 | コピートレードアカウントに証拠金を追加します。 |
| データを共有 | コピートレードのパフォーマンスを共有します。 |
| 再度コピートレード | 同じ対象ウォレットを基にもう一度コピートレードを作成します。 |
| 削除 | 停止済みまたは非アクティブなルールを削除します。 |

<Warning>
コピートレードの停止とルールの削除は別の操作です。停止はフォローを止めるためのもので、削除は通常非アクティブなレコードを整理するために使用します。
</Warning>
