사용자가 제품을 사용하기 시작하는 방법을 설명하세요. 가장 먼저 해야 할 일은 무엇인가요? 사전 요구 사항이 있나요?

퀵스타트는 사용자를 제로 상태에서 제품을 사용하는 상태로 이끌어야 합니다. 작은 성취감을 얻고 제품으로 무엇을 할 수 있는지 감을 잡게 됩니다.

## 사전 요구 사항

시작하기 전에 다음이 필요합니다:

- 요구 사항 1 (예: Node.js 18+, 무료 계정, API 키)
- 요구 사항 2 (예: 호환되는 기기, 호환되는 브라우저, 호환되는 운영 체제)

## 시작하기

<Steps>
  <Step title="설치">
    제품을 설치하거나 가입하는 방법을 설명하세요.

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="구성">
    첫 사용 전에 필요한 설정이나 구성을 설명하세요.

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="실행">
    사용자가 처음으로 동작을 확인하기 위해 수행하는 작업을 보여주세요.

    ```bash
    your-cli start
    ```
  </Step>
</Steps>

<Tip>
  사용자가 도움을 받을 수 있는 방법을 제공하세요. 지원 페이지 링크, 고객 지원 상담 채팅, 또는 제품 포럼이 될 수 있습니다.

  예: 도움이 필요하신가요? [support@yourcompany.com](mailto:support@yourcompany.com) 으로 문의하세요.
</Tip>
