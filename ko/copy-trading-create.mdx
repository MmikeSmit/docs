다음 두 위치에서 카피 트레이딩에 진입할 수 있습니다:

- 스마트 머니 목록에서 대상 지갑의 **카피** 버튼을 클릭합니다.
- 지갑 상세 페이지에서 거래 성과를 확인한 후 **카피** 버튼을 클릭합니다.

스마트 머니 또는 지갑 상세에서 진입하면 대상 지갑 주소가 자동으로 입력됩니다.  
마이페이지의 카피 트레이딩 페이지에서 생성할 때는 대상 지갑 주소를 직접 입력해야 합니다.

## 생성 절차

1. 대상 지갑의 카피 트레이딩 패널을 엽니다.
2. 현재 따라가는 지갑 주소를 확인합니다.
3. 카피 모드를 선택합니다: 스마트 카피 또는 즉시 카피.
4. 총 투자 금액을 입력합니다.
5. 사용 가능한 잔고, 최소 투자 금액, 발생 가능한 활성화 비용을 확인합니다.
6. **카피** 버튼을 눌러 규칙을 생성합니다.
7. 생성이 성공하면 **마이페이지 → 카피 트레이딩**으로 이동해 확인 및 관리합니다.

<Info>
카피 트레이딩은 생성 후 백엔드 서비스에서 비동기로 처리됩니다. 생성 성공이 모든 포지션의 즉각적인 동기화를 의미하지는 않으며, 페이지 데이터가 갱신되는 데 약간의 시간이 걸릴 수 있습니다.
</Info>
