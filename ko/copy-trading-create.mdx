다음 두 곳에서 카피 트레이딩에 진입하실 수 있습니다.

- 스마트 머니 리스트에서 대상 지갑의 **카피 트레이딩**을 클릭합니다.
- 지갑 상세 페이지에서 거래 성과를 확인하신 후 **카피 트레이딩**을 클릭합니다.

스마트 머니나 지갑 상세에서 진입하시면 리더 지갑 주소가 자동으로 입력됩니다.  
마이 페이지의 카피 트레이딩 페이지에서 생성하시는 경우에는 리더 지갑 주소를 수동으로 입력하셔야 합니다.

## 생성 흐름

1. 대상 지갑의 카피 트레이딩 패널을 엽니다.
2. 현재 따라가실 지갑 주소를 확인합니다.
3. 카피 트레이딩 모드를 선택합니다. 스마트 카피 또는 즉시 카피.
4. 총 투입을 입력합니다.
5. 사용 가능한 잔액, 최소 투입, 발생할 수 있는 활성화 수수료를 확인합니다.
6. **카피 트레이딩**을 클릭하여 규칙을 생성합니다.
7. 생성 성공 후 **마이 페이지 → 카피 트레이딩**에서 확인하고 관리합니다.

<Info>
카피 트레이딩 생성 후에는 백엔드 서비스에서 비동기로 처리됩니다. 생성 성공이 모든 포지션이 즉시 동기화 완료됨을 의미하지는 않으며, 페이지 데이터는 잠시 후 업데이트될 수 있습니다.
</Info>
