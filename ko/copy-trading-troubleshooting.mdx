---
title: "생성 불가의 일반적인 원인"
description: "카피 트레이딩 생성 실패 또는 버튼 비활성화 문제를 진단합니다."
mode: "wide"
---

**카피 트레이딩** 버튼을 사용하실 수 없다면, 보통 다음 원인 중 하나입니다.

| 원인 | 설명 |
| --- | --- |
| 지갑 미연결 | 카피 트레이딩을 생성하시려면 먼저 지갑을 연결하셔야 합니다. |
| 지갑 주소 무효 | 리더 지갑 주소는 유효한 EVM 주소여야 합니다. |
| 투입 금액이 너무 낮음 | 총 투입이 시스템 최소 요건보다 낮습니다. |
| 잔액 부족 | 사용 가능한 USDC가 투입 금액과 발생할 수 있는 활성화 수수료를 충당하기에 부족합니다. |
| 대상 지갑이 카피 트레이딩 불가 | 일부 지갑은 카피 트레이딩이 비활성화되어 있을 수 있습니다. |
| 서브 계정 수가 상한에 도달 | 카피 트레이딩 서브 계정의 상한에 도달했습니다. 새 카피 트레이딩을 생성하시려면 먼저 기존 카피 트레이딩 하나를 종료하셔야 합니다. |
