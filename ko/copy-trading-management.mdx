생성 성공 후 마이 페이지의 카피 트레이딩 페이지에 들어가시면 다음을 확인하실 수 있습니다.

- 카피 트레이딩 자산
- 7일 손익
- 사용 가능 증거금
- 손익
- 미실현 손익
- 현재 활성 카피 트레이딩 규칙
- 카피 트레이딩 이력

## 카피 트레이딩 관리

마이 페이지의 카피 트레이딩 페이지에서 이미 생성하신 카피 트레이딩 규칙을 관리하실 수 있습니다.

자주 사용되는 작업은 다음과 같습니다.

| 작업 | 설명 |
| --- | --- |
| 상세 | 카피 트레이딩 규칙의 상세를 확인합니다. |
| 종료 | 현재 카피 트레이딩을 중지합니다. |
| 증거금 추가 | 카피 트레이딩 계정에 증거금을 추가합니다. |
| 데이터 공유 | 카피 트레이딩 성과를 공유합니다. |
| 다시 카피 트레이딩 | 같은 대상 지갑을 기반으로 다시 카피 트레이딩을 생성합니다. |
| 삭제 | 종료되었거나 비활성 상태인 규칙을 삭제합니다. |

<Warning>
카피 트레이딩 종료와 규칙 삭제는 서로 다른 작업입니다. 종료는 따라가기를 중지하는 데 사용되며, 삭제는 보통 비활성 기록을 정리하는 데 사용됩니다.
</Warning>
