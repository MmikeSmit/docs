## 지갑 프로필 태그

지갑 카드의 아바타 옆에는 해당 지갑의 거래 특성을 빠르게 요약하기 위한 시스템 태그가 표시됩니다. 이 태그는 사용자가 직접 추가하는 즐겨찾기 라벨이 아니라, 지갑 데이터를 기반으로 생성된 프로필입니다.

자주 사용되는 태그는 다음 네 가지 그룹으로 이해할 수 있습니다.

### 계정 규모

현재 보유 포지션 규모에 따라 지갑의 크기를 구분하는 데 사용됩니다.

| 태그 | 의미 |
| --- | --- |
| 고래 | 현재 보유 포지션 규모가 매우 큰 지갑. |
| 상어 | 현재 보유 포지션 규모가 큰 지갑. |
| 돌고래 | 현재 보유 포지션 규모가 중간 수준인 지갑. |
| 작은 물고기 | 현재 보유 포지션 규모가 작은 지갑. |

### 종목 선호도

지갑이 주로 어떤 종류의 종목을 거래하는지 판단하는 데 사용됩니다.

| 태그 | 의미 |
| --- | --- |
| 메이저 | BTC, ETH 등 메이저 종목을 더 자주 거래함. |
| 알트코인 | 변동성이 높은 비메이저 종목을 더 자주 거래함. |
| TradFi | 전통 금융 관련 종목을 더 자주 거래함. |
| 종합 | 여러 종류를 거래하며 뚜렷한 단일 선호가 없음. |

### 방향성 선호도

지갑이 주로 롱을 잡는지, 숏을 잡는지, 아니면 롱·숏이 비교적 균형 잡혀 있는지를 판단하는 데 사용됩니다.

| 태그 | 의미 |
| --- | --- |
| 롱 편향 | 롱 포지션을 더 자주 보유함. |
| 숏 편향 | 숏 포지션을 더 자주 보유함. |
| 균형형 | 롱·숏 분포가 비교적 균형 잡혀 있음. |

### 거래 주기와 성과 마커

거래 템포와 최근 성과에 두드러진 특징이 있는지를 판단하는 데 사용됩니다.

| 태그 | 의미 |
| --- | --- |
| 초단타 | 더 짧은 빈도의 거래를 선호함. |
| 단기 | 더 짧은 보유 기간을 선호함. |
| 중장기 | 더 긴 보유 기간을 선호함. |
| 승률 헌터 | 승률 성과가 두드러짐. |
| 안정 성장 | 장기 성과가 비교적 안정적임. |
| 수익 머신 | 거래 활동성과 수익 성과가 두드러짐. |

예를 들어 스크린샷의 **균형형 / 롱 편향 / 단기**는 이 지갑이 프로필상 롱과 숏 분포가 비교적 균형 잡혀 있고, 최근 방향은 다소 롱 쪽이며, 거래 템포는 단기 위주임을 의미합니다. 먼저 이러한 태그로 자신의 카피 트레이딩 성향에 맞는지 판단한 다음, 상세 페이지로 들어가 포지션과 거래 내역을 확인할 수 있습니다.

### 계정 규모는 어떻게 판정하나요

계정 규모는 현재 보유 포지션 금액에 따라 구분됩니다.

| 태그 | 판정 기준 |
| --- | --- |
| 고래 | 포지션이 100만 USDC 초과 |
| 상어 | 포지션이 10만에서 100만 USDC |
| 돌고래 | 포지션이 5만에서 10만 USDC |
| 작은 물고기 | 포지션이 1만에서 5만 USDC |

이 태그는 주로 지갑 규모를 판단하는 데 사용됩니다. 규모가 클수록 일반적으로 실제 자금의 움직임을 더 잘 반영하지만, 규모가 작은 지갑은 소수의 거래만으로도 수익률이 크게 부풀려질 수 있습니다.

### 성과 마커는 어떻게 판정하나요

성과 마커는 최근 거래 성과가 두드러진 계정을 나타내는 데 사용됩니다.

| 태그 | 판정 기준 |
| --- | --- |
| 승률 헌터 | 승률 80% 초과, 그리고 30일 진입 횟수가 60회 초과 |
| 안정 성장 | 반년 이상 안정적으로 수익을 내고, 30일 진입 횟수가 30회 초과 |
| 수익 머신 | 승률 60% 초과, 그리고 30일 진입 횟수가 300회 초과 |

이러한 마커는 지갑이 해당 데이터 조건을 충족했음을 의미할 뿐이며, 앞으로도 계속 수익을 낼 것을 보장하지는 않습니다. 카피 트레이딩 여부를 판단할 때는 현재 포지션, 손실폭, 레버리지, 거래 종목도 함께 확인해야 합니다.
